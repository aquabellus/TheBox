        |--BoxDump.d/...
        |--The-Box
            |--/../
            |--log/
            |--helper/
            |--src/
            |--aquabot.py
            |--aquasetup.py
            |--aquastart.py
            |--setup.json
            |--setup.sh
            |--..
            
    *BoxDump.d will automatically generated when you run aquastart.py
